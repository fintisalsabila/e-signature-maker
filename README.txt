# E-Signature Maker with Download Using HTML, CSS, and JavaScript
The **E-Signature Maker with Download** is a simple yet efficient tool for creating and downloading digital signatures. Ideal for contracts, forms, and other documents, this web application allows you to draw, adjust, and download your signature as a PNG file. Designed to be user-friendly and intuitive, this tool is suitable for both personal and professional use.

---

### Project Features
- **Create E-Signatures**: Draw your signature using a mouse or touch device on a canvas.
- **Clear Canvas**: Reset the canvas to start a fresh drawing.
- **Undo Strokes**: Undo individual strokes to correct any mistakes.
- **Download Signature**: Save your completed signature as a PNG file, ready for use in digital documents.

---

### Technologies Used
- **HTML**: For structuring the application.
- **CSS**: Custom fonts and responsive design for a visually appealing interface.
- **JavaScript**: Handles the drawing, undo, clear, and download functionalities.

---

### How to Use
1. **Draw Signature**: Click and hold the mouse (or touch and drag on mobile) to draw your signature on the canvas.
2. **Undo a Stroke**: Click the "Undo" button to remove the last stroke if needed.
3. **Clear the Canvas**: Click the "Clear" button to erase everything and start over.
4. **Download the Signature**: Once you're satisfied, click the "Download" button to save the signature as a PNG file on your device.

---

### Installation Guide
1. **Download the Source Code**: Download the provided zip file.
2. **Extract the Files**: Locate and extract the downloaded zip file.
3. **Open the Application**: Find the `index.html` file in the extracted folder.
4. **Run in Browser**: Open `index.html` in a web browser (e.g., Chrome, Firefox).
